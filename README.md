## How to Run

Frog Game is built with **Godot**.

1. Install [Godot 4.x](https://godotengine.org/).
2. Clone this repository:
   ```bash
   git clone https://github.com/webbqed/frog-game.git
   cd frog-game
   ```
3. Open the project in Godot.
4. Press **Play** to run the game.
